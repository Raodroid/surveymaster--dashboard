# NOTE: Access to client-dashboard folder before installation and running

```bash
$ cd client-dashboard
```

### ==> Installation <==

```bash
$ yarn install
```

### ==> Running the app <==

# Step 2: Create an ENV file based on .env.example file before running

```bash
# Local
$ .env

# Development
$ .env.dev

# Staging
$ .env.dev
```

# Step 3: Run command

```bash
# Local
$ yarn start

# Development
$ yarn start:dev

# Staging
$ yarn start:stg
```
